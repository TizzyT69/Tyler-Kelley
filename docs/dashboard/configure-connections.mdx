## Group Connections

Group connections are a feature that allows you to link multiple login connections to access the
same user address across different connections. Note that group keys differ from individual
connection keys, and before proceeding with group connections, you need to create single connections
that is supported for group connections.

<div style={{ flexBasis: "300px", flexGrow: "1", textAlign: "center" }}>
  <img
    src={AuthenticationGroupConnectionsCreate}
    style={{ alignSelf: "center", maxWidth: "100%" }}
    alt="Authentication Group Connections Create"
  />
</div>

Only the possible connections are shown in the dropdown. Check the table to see which connections
are supported for group connections.

<div style={{ flexBasis: "300px", flexGrow: "1", textAlign: "center" }}>
  <img
    src={AuthenticationGroupConnectionsSelect}

this grouped connection will not migrate users from the individual social or custom connections.
