While using the Web3Auth Unity SDK, you get a private key that can be used with libraries like
[`Nethereum`](https://nethereum.com/) to make [Avalanche](https://www.avax.network/) blockchain
calls like getting the user's `account`, fetching `balance`, `sign transaction`, `send transaction`,
`read` from and `write` to the smart contract, etc. We have highlighted a few here to get you
started quickly on that.

## Installation

To interact with the Avalanche blockchain, you can use the Nethereum library with Web3Auth.

  ]}

<TabItem value="npm">

```bash
npm install --save nethereum.unity
```

</TabItem>
<TabItem value="openupm">

```bash
openupm add nethereum.unity
```

</TabItem>
</Tabs>

## Initializing Provider

Using `eip155` as `chainNamespace` while initializing `web3auth` will provide a private key that can
be used to initialize the Nethereum library and make blockchain calls.
