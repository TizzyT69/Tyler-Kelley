## How it works?

When integrating Web3Auth with Auth0 using the implicit flow:

![Auth0 Implicit Flow](/guides/auth0/auth0-spa-flow.png)

In this flow, Web3Auth handles the Auth0 authentication directly. The user is redirected to Auth0
for authentication, and upon successful login, Web3Auth receives the authentication credentials and
generates the user's wallet.

</TabItem>
<TabItem value="jwt">

When integrating Web3Auth with Auth0 using the JWT flow:

![Auth0 JWT Flow](/guides/auth0/auth0-rwa-flow.png)

In this flow, your application handles the Auth0 authentication process first. After successful
authentication, you obtain the JWT token (ID token) from Auth0 and pass it to Web3Auth, which then
validates the token and generates the user's wallet.

</TabItem>

</Tabs>

- When a user logs in with Auth0, Auth0 sends a JWT `id_token` to the app, either directly to
  Web3Auth (implicit flow) or through your application (JWT flow).

- On successful validation of the JWT token, Web3Auth will generate a private key for the user, in a
  self-custodial way, resulting in easy onboarding for your user to the application.

## Using the Web3Auth Plug and Play No Modal SDK

To use Web3Auth with Auth0, you need to set up your React application with the Web3Auth SDK. This
guide will walk you through setting up both implicit and JWT authentication methods.

<SetupBaseProject />

### Installation

Install the necessary dependencies for your React application:

```bash
# Core Web3Auth packages
npm install @web3auth/modal

# For Auth0 JWT flow
npm install @auth0/auth0-react

# For blockchain interactions
npm install wagmi viem
```

### Setting up Web3Auth Provider

First, create a configuration file for Web3Auth (e.g., `web3authContext.tsx`):

```tsx
import { WEB3AUTH_NETWORK, type Web3AuthOptions } from "@web3auth/modal";

// Get your client ID from https://dashboard.web3auth.io
const clientId = import.meta.env.VITE_WEB3AUTH_CLIENT_ID || "";

// Instantiate SDK
const web3AuthOptions: Web3AuthOptions = {
  clientId,
  web3AuthNetwork: WEB3AUTH_NETWORK.SAPPHIRE_MAINNET,
};

const web3AuthContextConfig = {
  web3AuthOptions,
};

export default web3AuthContextConfig;
```

Then, set up the Web3Auth provider in your application entry point (`index.tsx` or similar):

```tsx
import ReactDOM from "react-dom/client";
// Setup Web3Auth Provider
import { Web3AuthProvider } from "@web3auth/modal/react";
import web3AuthContextConfig from "./web3authContext";
// Setup Wagmi Provider for blockchain interactions
import { WagmiProvider } from "@web3auth/modal/react/wagmi";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";

import App from "./App";

const queryClient = new QueryClient();

const root = ReactDOM.createRoot(document.getElementById("root") as HTMLElement);
root.render(
  // Setup Web3Auth Provider
  <Web3AuthProvider config={web3AuthContextConfig}>
    {/* Setup Wagmi Provider */}
    <QueryClientProvider client={queryClient}>
      <WagmiProvider>
        <App />
      </WagmiProvider>
    </QueryClientProvider>
  </Web3AuthProvider>,
);
```

### Authentication Implementation

You can implement Auth0 authentication with Web3Auth using either the implicit flow or the JWT flow:

<Tabs

<TabItem value="implicit">

For the implicit flow, Web3Auth handles the Auth0 authentication directly:

```tsx
import { useWeb3AuthConnect, useWeb3AuthDisconnect, useWeb3AuthUser } from "@web3auth/modal/react";
import { WALLET_CONNECTORS, AUTH_CONNECTION } from "@web3auth/modal";
import { useAccount } from "wagmi";

function App() {
  const { connectTo, isConnected, connectorName } = useWeb3AuthConnect();
  const { disconnect } = useWeb3AuthDisconnect();
  const { userInfo } = useWeb3AuthUser();
  const { address } = useAccount();

  const loginWithAuth0 = async () => {
    await connectTo(WALLET_CONNECTORS.AUTH, {
      authConnection: AUTH_CONNECTION.CUSTOM,
      authConnectionId: "w3a-auth0-demo", // Your Auth Connection ID from Web3Auth Dashboard
    });
  };

  // UI implementation...
}
```

</TabItem>
<TabItem value="jwt">

For the JWT flow, you need to integrate Auth0's authentication SDK and pass the token to Web3Auth:

```tsx
import { useWeb3AuthConnect, useWeb3AuthDisconnect, useWeb3AuthUser } from "@web3auth/modal/react";
import { WALLET_CONNECTORS, AUTH_CONNECTION } from "@web3auth/modal";
import { useAccount } from "wagmi";
import { useAuth0 } from "@auth0/auth0-react";

function App() {
  const { connectTo, isConnected, connectorName } = useWeb3AuthConnect();
  const { disconnect } = useWeb3AuthDisconnect();
  const { userInfo } = useWeb3AuthUser();
  const { address } = useAccount();
  const { getIdTokenClaims, loginWithPopup, logout } = useAuth0();

  const loginWithAuth0 = async () => {
    try {
      await loginWithPopup();
      const idToken = (await getIdTokenClaims())?.__raw.toString();
      if (!idToken) {
        throw new Error("No id token found");
      }
      await connectTo(WALLET_CONNECTORS.AUTH, {
        authConnectionId: "w3a-auth0-demo", // Your Auth Connection ID from Web3Auth Dashboard
        authConnection: AUTH_CONNECTION.CUSTOM,
        idToken,
        extraLoginOptions: {
          isUserIdCaseSensitive: false,
        },
      });
    } catch (err) {
      console.error(err);
    }
  };

  // UI implementation...
}
```

For the JWT flow, you also need to set up Auth0 in your main index file:

```tsx
import { Auth0Provider } from "@auth0/auth0-react";

// ...inside your render method
<Auth0Provider
  domain={import.meta.env.VITE_AUTH0_DOMAIN || ""}
  clientId={import.meta.env.VITE_AUTH0_CLIENT_ID || ""}
  authorizationParams={{
    redirect_uri: window.location.origin,
  }}
>
  <Web3AuthProvider config={web3AuthContextConfig}>{/* ... */}</Web3AuthProvider>
</Auth0Provider>;
```

### Getting User Info and Logging Out

```tsx
// Get user info
const { userInfo } = useWeb3AuthUser();
console.log("User info", userInfo);

// Logout
const { disconnect } = useWeb3AuthDisconnect();
await disconnect();

// With Auth0 JWT flow, also log out from Auth0
const { logout } = useAuth0();
logout({
  logoutParams: {
    returnTo: window.location.origin,
  },
});
```

## Interacting with the Blockchain

With Web3Auth v10, blockchain interactions are simplified using the wagmi hooks:

```tsx
// Example of getting user balance
import { useAccount, useBalance } from "wagmi";
import { formatUnits } from "viem";

export function Balance() {
  const { address } = useAccount();
  const { data, isLoading, error } = useBalance({ address });

  return (
    <div>
      <h2>Balance</h2>
      <div>
        {data?.value !== undefined && `${formatUnits(data.value, data.decimals)} ${data.symbol}`}
        {isLoading && "Loading..."}
        {error && "Error: " + error.message}
      </div>
    </div>
  );
}

// Example of switching networks
import { useChainId, useSwitchChain } from "wagmi";

export function SwitchChain() {
  const chainId = useChainId();
  const { chains, switchChain, error } = useSwitchChain();

  return (
    <div>
      <h2>Switch Chain</h2>
      <h3>Connected to {chainId}</h3>
      {chains.map((chain) => (
        <button
          disabled={chainId === chain.id}
          key={chain.id}
          onClick={() => switchChain({ chainId: chain.id })}
          type="button"
          className="card"
        >
          {chain.name}
        </button>
      ))}
      {error?.message}
    </div>
  );
}
```
